# 黑名单系统插件使用文档

## 简介

黑名单系统是一款用于QQ群聊和私聊的消息管理插件，支持：

- **用户拉黑**：将恶意用户加入黑名单，其消息将被自动忽略
- **群聊屏蔽**：屏蔽整个群（管理员除外）
- **机器人闭嘴**：临时让机器人在指定群停止响应
- **灵活配置**：支持按群独立黑名单、全局黑名单、定时解封、自定义理由等

插件内置LLM工具（函数调用），AI可根据对话内容自主拉黑恶意用户或闭嘴。

---

## 安装

1. 将插件文件夹放入 `AstrBot/plugins/` 目录
2. 重启AstrBot或执行热重载
3. 在插件配置中根据需要修改 `data/plugins/黑名单系统/_conf_schema.json` 或通过WebUI配置

---

## 配置说明

配置文件位于 `data/plugins/黑名单系统/_conf_schema.json`，也可在AstrBot WebUI中直接修改。

| 配置项 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| `黑名单群聊` | list | 被完全屏蔽的群号列表，该群所有消息（管理员除外）将被忽略 | `[]` |
| `黑名单用户` | list | 黑名单条目，格式见下方说明 | `[]` |
| `包括指令` | bool | 是否拦截黑名单用户发送的指令（如`/help`）。`true`表示连指令也拦截，`false`表示允许执行指令 | `true` |
| `闭嘴列表` | list | 机器人闭嘴的结束时间戳（格式：`群号:时间戳`），**请勿手动修改** | `[]` |
| `显示日志` | bool | 是否在控制台打印被黑名单拦截的日志 | `true` |

### 黑名单用户格式

每个条目的格式为：  
`用户ID | 结束时间戳 | 用户昵称 | 拉黑理由 | 群号 | 操作者`

- `用户ID`：必填，QQ号
- `结束时间戳`：Unix时间戳（秒），不填默认为2131年（永久）
- `用户昵称`：可选，便于识别
- `拉黑理由`：可选
- `群号`：`"全局"` 表示全局黑名单，否则为指定群号
- `操作者`：操作人QQ号或标识（如 `"管理员"`、`"llm"`）

**示例**：  
`123456789|5102444800|张三|辱骂群友|678910|管理员`

> 提示：通常无需手动编辑该列表，使用拉黑指令即可自动添加。

## 指令列表

### 通用指令（所有人可用）

| 指令 | 说明 |
|------|------|
| `查看黑名单` / `黑名单` / `黑名单列表` | 查看当前群的拉黑列表（仅显示本群条目） |
| `闭嘴` @机器人 | 让机器人在本群闭嘴一段时间（需要时间参数，如 `闭嘴 30分钟`） |
| `开嘴` @机器人 | 解除本群的闭嘴状态 |

> **注意**：`闭嘴`和`开嘴`必须**艾特机器人**才生效。

### 管理员指令（群主/管理员/配置的admin）

| 指令 | 格式 | 说明 |
|------|------|------|
| `拉黑` | `拉黑 @用户 10分钟 理由`<br>`拉黑 回复某条消息 1小时`<br>`拉黑 123456 理由` | 将用户加入当前群的黑名单。可自动从艾特或回复中获取用户ID。支持时间单位：分钟/小时/天/月/年，支持中文数字 |
| `解除拉黑` | `解除拉黑 @用户`<br>`解除拉黑 123456`<br>`解除拉黑`（无参数时清空当前群所有黑名单） | 从当前群黑名单中移除用户 |
| `全局拉黑` | `/全局拉黑 123456 30天 理由` | 将用户加入全局黑名单（所有群和私聊生效） |
| `全局解除拉黑` | `/全局解除拉黑 123456` | 从全局黑名单移除用户 |
| `清空黑名单` | `/清空黑名单`<br>`/清空黑所有名单`<br>`/清空黑名单 123456` | 无参数：清空当前群黑名单；`all`：清空所有黑名单；传入群号：清空指定群黑名单 |
| `屏蔽此群` / `屏蔽该群` / `屏蔽这个群` | `/屏蔽此群` 或 `/屏蔽此群 群号` | 屏蔽整个群（管理员除外），可用于临时关闭某个群的机器人响应 |
| `取消屏蔽` / `取消屏蔽此群` | `/取消屏蔽` 或 `/取消屏蔽 群号` | 解除群屏蔽 |
| `群拉黑` | `/群拉黑 群号 用户ID 时长(分钟)` | 远程拉黑指定群的用户（无需在目标群内） |
| `群解除拉黑` | `/群解除拉黑 群号 用户ID` | 远程解除指定群的黑名单 |

#### 时间参数说明

- 支持单位：`分钟`/`分`、`小时`/`时`、`天`、`个月`/`月`、`年`
- 数字支持阿拉伯数字和中文数字（如 `十`、`三十`）
- 示例：`30分钟`、`2小时`、`1天`、`一周`、`两个月`、`1年`

#### 拉黑理由与名字

- 可在指令末尾使用 `理由:xxx` 指定理由
- 使用 `名字:xxx` 指定显示名称（不指定则自动获取）
- 示例：`拉黑 123456 1小时 理由:多次发广告 名字:张三`

## LLM工具（函数调用）

插件为AI提供了以下工具，AI可在对话中自主调用：

| 工具名 | 说明 | 参数 |
|--------|------|------|
| `add_to_blacklist` | 将恶意用户拉黑 | `用户ID`（可选，默认为当前发送者）、`时长`（分钟，-1表示永久）、`理由`（可选） |
| `list_blacklist` | 查看当前群的黑名单 | 无 |
| `shut_up` | 让机器人闭嘴一段时间 | `时长`（分钟，默认40） |

> **使用场景**：当用户持续辱骂、发送垃圾信息时，AI可主动调用 `add_to_blacklist` 拉黑该用户。

## 使用示例

### 示例1：拉黑当前艾特的用户

```
/拉黑 @小明 30分钟 理由:恶意刷屏
```
系统回复：  
✅ 已添加  
「小明(123456) 群'678910'」到黑名单！  
时长：30分钟  
理由：恶意刷屏

### 示例2：根据回复消息拉黑

```
回复某条消息后发送：/拉黑 1小时
```
自动拉黑被回复消息的发送者。

### 示例3：全局拉黑

```
/全局拉黑 987654 7天 理由:广告机器人
```

### 示例4：让机器人闭嘴

```
@机器人 闭嘴 2小时
```
机器人回复：`我将闭嘴2小时`，期间该群所有消息不会触发机器人响应。

### 示例5：查看黑名单

```
/查看黑名单
```
输出示例：
```
有3个用户在黑名单，格式：
用户ID | 剩余时间 | 用户名 | 拉黑理由

123456 | 25分钟 | 张三 | 辱骂
789012 | 永久 | 李四 | 广告
```

### 示例6：清空当前群黑名单

```
/清空黑名单
```

## 注意事项

1. **权限要求**：拉黑/解除拉黑等管理操作需要发送者是**群管理员/群主**或在AstrBot配置的 `admins_id` 中。
2. **黑名单优先级**：全局黑名单 > 群黑名单；私聊仅受全局黑名单控制。
3. **指令前缀**：默认使用系统配置的 `wake_prefix`（通常为 `/` 或空），示例中使用了 `/`，请根据你的实际配置调整。
4. **私聊处理**：插件仅对已拉黑的私聊用户生效，不会自动响应非黑名单私聊。
5. **时间精度**：拉黑时长以分钟为单位，实际解封时间可能会有分钟级误差。
6. **闭嘴功能**：`闭嘴`指令必须艾特机器人，且只有群内成员可触发；`开嘴`同样需要艾特机器人。
7. **数据持久化**：所有黑名单和闭嘴记录会保存到配置文件，重启机器人后依然生效。

## 常见问题

**Q：拉黑后用户还能发指令吗？**  
A：取决于配置项 `包括指令`。若设为 `false`，用户仍可使用指令（如 `/help`）；若设为 `true`，所有消息均被拦截。

**Q：如何解封永久的黑名单？**  
A：使用 `解除拉黑` 指令即可。

**Q：为什么我屏蔽了某个群，管理员还能发消息？**  
A：设计如此，群管理员和机器人的主人（配置的admin）不受群屏蔽影响。

**Q：闭嘴时间到了会自动开嘴吗？**  
A：是的，计时结束后机器人自动恢复响应。

**Q：能否在WebUI中管理黑名单？**  
A：可以，黑名单列表在配置项中可见，但建议使用指令操作，避免格式错误。
